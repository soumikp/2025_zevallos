# 2025_zevallos: repo for collaboration with Zevallos et al. (CHERP)

This repository contains the following directories:

1. **[/analysis](https://github.com/soumikp/2025_zevallos/tree/main/analysis)**: Contains generated PDF plots and analysis outputs related to surgery times, costs, and exploratory data analysis.

2. **[/code](https://github.com/soumikp/2025_zevallos/tree/main/code)**: Contains R scripts, R Markdown files, and HTML reports for simulators, helpers, and EDA related to the report.

3. **[/data](https://github.com/soumikp/2025_zevallos/tree/main/data)**: Contains raw data files, including CSV and Excel files (e.g., surgery records, percentage data).

4. **[/documents](https://github.com/soumikp/2025_zevallos/tree/main/documents)**: Contains various word documents, presentations, and PDFs including proposals, meeting notes, and research plans.

5. **[/literature](https://github.com/soumikp/2025_zevallos/tree/main/literature)**: Contains relevant literature and research articles in PDF format.
